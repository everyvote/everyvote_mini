How to Add an Election

You must be a moderator for a group in order to create an election.

If you are a moderator, then open your group's page, and the Add Election button will be active.

How to Edit an Election

You must be a moderator for a group in order to edit an election.

If you are a moderator, then open the election page you want to edit, and the Edit Election button will be active.

How to Edit a Group

You must be a moderator for a group in order to edit a group.

If you are a moderator, then open the group page you want to edit, and the Edit Group button will be active.
